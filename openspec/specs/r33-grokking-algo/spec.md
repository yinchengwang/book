# R33 Grokking 算法篇 — 规格定义

## 概述

Grokking 算法篇提供 16 张算法学习卡片，覆盖面试高频算法模式。每张卡片包含 C11 实现的代码示例、可构建的 Makefile、README 和 NOTES 工程对照。

## 目录结构

```
learning/scaffold/grokking/
├── big_o/                     # 大 O 分析
├── arrays/                    # 数组技巧
├── linked_lists/              # 链表技巧
├── stacks_queues/             # 栈与队列
├── trees_graphs/              # 树与图
├── heap/                      # 堆应用
├── recursion/                 # 递归思想
├── sorting/                   # 排序应用
├── searching/                 # 搜索技巧
├── dynamic_programming/       # 动态规划
├── greedy/                    # 贪心选择
├── graph/                     # 图算法
├── intervals/                 # 区间处理
├── math/                      # 数学技巧
├── bit_manipulation/          # 位运算
└── two_pointers/              # 双指针
```

## 卡片规范

每张卡片包含 4 个文件：

| 文件 | 用途 | 格式要求 |
|------|------|----------|
| `main.c` | 核心代码 | C11（`-std=c11`），中文注释，~100-140 行 |
| `Makefile` | 构建 | gcc 编译，all/run/clean 目标 |
| `README.md` | 卡片说明 | 标题、简介、运行方法 |
| `NOTES.md` | 工程对照 | ≥100 字，算法→工程映射 |

## 覆盖模式

| 序号 | 模式 | 核心算法 |
|------|------|----------|
| 1 | 复杂度分析 | O(1)/O(log n)/O(n)/O(n log n)/O(n²)/O(2ⁿ) |
| 2 | 数组 | 两数之和、旋转数组、滑动窗口 |
| 3 | 链表 | 反转、环检测、合并、快慢指针 |
| 4 | 栈与队列 | 括号匹配、最小栈、约瑟夫环 |
| 5 | 树与图 | 层序遍历、对称树、路径总和 |
| 6 | 堆 | Top-K、动态中位数、K 路合并 |
| 7 | 递归 | 阶乘、斐波那契、回溯模板 |
| 8 | 排序 | 合并区间、快速选择、桶排序 |
| 9 | 搜索 | 二分查找、旋转数组、边界搜索 |
| 10 | 动态规划 | 爬楼梯、编辑距离、股票买卖 |
| 11 | 贪心 | 活动选择、哈夫曼编码、加油站 |
| 12 | 图 | 克隆图、课程表、岛屿数量 |
| 13 | 区间 | 合并区间、插入区间、会议室 |
| 14 | 数学 | 筛法求质、随机数、蓄水池采样 |
| 15 | 位运算 | set/clear/toggle、popcount、交换奇偶位 |
| 16 | 双指针 | 三数之和、接雨水、环形链表入口 |

## 状态管理

- `statuses.json`: 16 个新 ID（`big_o` ~ `two_pointers`），状态 `done`
- `r33-progress.md`: 16 行审计日志

## 工程设计

- 语言: C11（`-std=c11`）
- 编译器: gcc，标志 `-Wall -Wextra`
- 无外部依赖，gcc 标准库即可编译
