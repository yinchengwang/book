# games-level-progression

关卡进度系统规范，支持章节、关卡解锁和星星评价。

## Purpose

该规格定义游戏的关卡进度系统，包括数独和消消乐的关卡设计、章节解锁和星星评价。

## ADDED Requirements

### Requirement: 数独关卡系统

#### Scenario: 数独章节列表
- **WHEN** 用户打开数独游戏
- **THEN** 显示章节选择界面：
  - 第 1 章：入门（关卡 1-10）
  - 第 2 章：初级（关卡 11-20）
  - 第 3 章：中级（关卡 21-30）
  - 第 4 章：高级（关卡 31-40）
  - 第 5 章：专家（关卡 41-50）

#### Scenario: 关卡解锁条件
- **WHEN** 完成前一章所有关卡
- **THEN** 自动解锁下一章节

#### Scenario: 关卡内难度选择
- **WHEN** 进入某个关卡
- **THEN** 可选择简单/中等/困难三种难度

#### Scenario: 星星评价系统
- **WHEN** 完成数独关卡
- **THEN** 根据用时给予 1-3 星：
  - 用时 < 2 分钟：3 星
  - 用时 2-5 分钟：2 星
  - 用时 > 5 分钟：1 星

### Requirement: 消消乐关卡系统

#### Scenario: 消消乐章节列表
- **WHEN** 用户打开消消乐游戏
- **THEN** 显示章节选择界面：
  - 第 1-3 章：普通关（关卡 1-27）
  - 第 4-5 章：困难关（关卡 28-45）
  - 第 6 章：BOSS 关（关卡 46-50）

#### Scenario: 章节解锁条件
- **WHEN** 累计获得足够星星
- **THEN** 解锁下一章节：
  - 第 1 章：0 星（初始解锁）
  - 第 2 章：15 星
  - 第 3 章：30 星
  - 第 4 章：50 星
  - 第 5 章：75 星
  - 第 6 章：100 星

#### Scenario: 关卡目标系统
- **WHEN** 进入消消乐关卡
- **THEN** 显示关卡目标：
  - 目标分数（⭐）
  - 收集特殊元素（🔑🪙🐦）

#### Scenario: 消消乐星星评价
- **WHEN** 完成消消乐关卡
- **THEN** 根据剩余步数比例给予 1-3 星：
  - 剩余步数 >= 50%：3 星
  - 剩余步数 >= 25%：2 星
  - 剩余步数 < 25%：1 星

### Requirement: 进度持久化

#### Scenario: 保存关卡进度
- **WHEN** 完成关卡获得星星
- **THEN** 自动保存到本地存储

#### Scenario: 恢复关卡进度
- **WHEN** 重新打开游戏
- **THEN** 显示各章节解锁状态和星星数

### Requirement: 提示次数限制

#### Scenario: 数独提示次数
- **WHEN** 开始数独游戏
- **THEN** 获得 3 次提示机会

#### Scenario: 提示次数耗尽
- **WHEN** 提示次数为 0 时点击提示
- **THEN** 显示"提示次数已用完"提示，无数字填充
