# S23 Spec —— Project Status Badges

## 1. 5 badges 契约

| Badge | URL | 含义 |
|---|---|---|
| Repo Size | shields.io/github/repo-size | 仓库 kLOC |
| Commits | shields.io/github/commit-activity | 30 天提交数 |
| OpenSpec Changes | shields.io/badge/OpenSpec-19-blue | 归档 change 计数 |
| Engineering Coverage | shields.io/badge/Cov-TBD-lightgrey | 0.0%（待 baseline 生成） |
| Learning CTest | shields.io/badge/Tests-158-brightgreen | 158 |

## 2. 静态 vs 动态

S23 使用 shields.io 静态 badge URL（**纯字符串 URL**），不需任何 server 端生成。

## 3. 不做

- ❌ 自动更新
- ❌ shields.io 自托管
- ❌ 真实数字生成（待 Phase 2）
