# ds-bit 规格

## 概述

位操作是算法中的重要技巧。本规格定义常用位操作技巧的实现及可视化文档要求。

## 目录结构

```
src/ds/bit/
├── README.md           # 详细文档（含 Mermaid 可视化）
└── problems/          # LeetCode 题目
```

## 实现功能

| 功能 | 函数名 | 说明 |
|------|--------|------|
| 获取某位 | `get_bit()` | 获取第 n 位的值 |
| 设置某位 | `set_bit()` | 将第 n 位设为 1 |
| 清除某位 | `clear_bit()` | 将第 n 位设为 0 |
| 切换某位 | `toggle_bit()` | 切换第 n 位 |
| 判断奇偶 | `is_odd()` | 奇数返回 true |
| 统计位数 | `count_bits()` | 统计 1 的个数 |
| 最低设置位 | `lowest_set_bit()` | 获取最低的 1 位 |

## README.md 文档要求

1. 二进制位布局可视化
2. 每个操作的具体示例（如 `5 = 101`，设置第 0 位后变成 `101 = 5`）
3. 位操作应用场景说明
