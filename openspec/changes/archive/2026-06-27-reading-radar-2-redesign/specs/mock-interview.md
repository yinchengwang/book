# Mock Interview 模拟面试

## Overview

模拟面试功能提供 AI 驱动的面试练习，预留接口支持未来接入 LLM。

## Features

### F12.1 面试配置

**描述**：配置模拟面试参数

**配置项**：
| 配置 | 选项 | 默认 |
|------|------|------|
| 岗位方向 | 后端 / 前端 / 全栈 / 算法 | 后端 |
| 技术栈 | Python / Java / Go / C++ | Python |
| 面试类型 | 技术面 / HR面 / 行为面 | 技术面 |
| 难度 | 初级 / 中级 / 高级 | 中级 |
| 题量 | 3题 / 5题 / 10题 | 5题 |

### F12.3 AI 追问

**描述**：AI 根据回答进行追问

**追问逻辑**：
1. 检测回答关键词
2. 生成针对性追问
3. 提供参考回答提示

**追问示例**：
```
检测到你提到了「缓存」，AI 追问：
"你提到了使用 Redis 缓存，能具体说说
缓存的过期策略是如何设计的吗？"
```

### F12.5 题库选择

**描述**：选择面试题目类型

**题目分类**：
| 类别 | 说明 | 示例 |
|------|------|------|
| 算法题 | 手写代码 | LRU、TOP-K、二分 |
| 系统设计 | 架构设计 | 短链接、秒杀系统 |
| 项目深挖 | 项目细节 | 项目亮点、难点 |
| 场景题 | 实际问题 | 故障排查、优化 |
| 行为面 | 软技能 | 团队协作、职业规划 |

---

### F12.6 面试记录

**描述**：保存历史面试记录

**记录内容**：
```typescript
interface MockInterviewRecord {
  id: string;
  date: string;
  position: string;
  config: InterviewConfig;
  questions: {
    question: string;
    answer: string;
    aiFollowup?: string[];
    rating?: number;
  }[];
  overallRating?: number;
  feedback?: string;
  createdAt: string;
}
```

## API 预留（云端）

```typescript
// 模拟面试 API（预埋）
interface MockInterviewAPI {
  // 生成面试问题
  generateQuestions(config: InterviewConfig): Promise<Question[]>;
  
  // 生成 AI 追问
  generateFollowup(question: string, answer: string): Promise<string>;
  
  // 评估回答
  evaluateAnswer(question: string, answer: string): Promise<Evaluation>;
  
  // 生成报告
  generateReport(records: InterviewRecord[]): Promise<Report>;
}
```

## Technical Notes

- 当前版本：本地题库 + 预留 API 接口
- AI 功能：需接入 LLM（OpenAI/DeepSeek）
- 语音输入：Web Speech API
- 数据存储：localStorage

## Status

> ⚠️ 此功能为**预埋功能**，MVP 阶段仅提供本地题库练习，AI 面试官功能待后续接入 LLM API。
