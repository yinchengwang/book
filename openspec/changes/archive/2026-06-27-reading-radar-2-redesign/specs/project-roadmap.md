### F13.5 关联知识

**描述**：项目与知识点的关联

**关联方式**：
```
项目：RESTful API 服务
  │
  ├── 技术点：FastAPI
  │     └── 关联题目：15 道
  │
  ├── 技术点：MySQL
  │     └── 关联题目：20 道
  │
  └── 技术点：Redis
        └── 关联题目：8 道
```

**学习联动**：
- 完成项目 → 自动标记相关知识点
- 学习相关题库 → 项目进度提示

## Data Models

```typescript
// 项目
interface Project {
  id: string;
  level: number;
  title: string;
  description: string;
  deliverable?: string;
  tech: string[];
  features: ProjectFeature[];
  duration: string;
  status: 'planned' | 'in_progress' | 'completed';
  progress: number;
  domain: string;
  knowledgeLinks: string[];  // 关联知识点 ID
  createdAt: string;
  updatedAt: string;
}

// 功能点
interface ProjectFeature {
  id: string;
  name: string;
  completed: boolean;
  description?: string;
}
```

## Technical Notes

- 项目管理：简单的状态机 + 进度追踪
- 知识关联：基于技术栈标签匹配
- 导出：项目文档导出为 Markdown
