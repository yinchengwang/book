# Gap Analysis 差距分析

## Overview

差距分析帮助用户识别知识薄弱点，推荐优先学习方向。

## Features

### F7.1 领域掌握度分析

### F7.3 智能推荐

**描述**：基于分析结果的个性化推荐

**推荐策略**：
1. **优先级排序**：
   - 高权重领域优先
   - 低掌握度优先
   - 高频考点优先

2. **学习路径推荐**：
   ```
   今日推荐：
   1. Attention 机制 ← 当前薄弱点
   2. Multi-Head Attention ← 依赖上一项
   3. Transformer 架构 ← 综合应用
   ```

3. **预估时间**：
   - 每个知识点预估学习时间
   - 今日可完成度

---

### F7.4 诊断自测

**描述**：快速诊断薄弱领域

**流程**：
1. 从每个领域抽取 1-2 题
2. 覆盖不同难度
3. 完成后生成诊断报告

**报告内容**：
```
┌─────────────────────────────────────────────────────┐
│                 诊断完成 ✅                         │
│                                                     │
│ 测试范围：5 个领域，12 道题                        │
│ 正确率：7/12 (58%)                                │
│                                                     │
│ ─────────────────────────────────────────────────  │
│                                                     │
│ 薄弱领域：                                         │
│ 1. LLM技术栈 - 正确率 33%                         │
│ 2. AI基础理论 - 正确率 50%                         │
│                                                     │
│ 建议优先学习：                                     │
│ • Attention 机制                                  │
│ • RLHF 对齐技术                                   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### F7.5 学习目标设定

**描述**：设定学习目标，对比差距

**目标配置**：
```typescript
interface LearningGoal {
  domain: string;        // 领域
  targetScore: number;   // 目标分数
  deadline: string;     // 截止日期
}
```

**差距视图**：
```
目标：C语言 掌握度 80%
当前：42%
差距：38%
预计：每日学习 2 题，需 19 天
```

## Data Models

```typescript
// 领域分析
interface DomainAnalysis {
  key: string;
  name: string;
  color: string;
  total: number;
  viewed: number;
  mastered: number;
  correctTotal: number;
  score: number;          // 掌握度百分比
}

// 薄弱点
interface WeakPoint {
  topicId: string;
  domain: string;
  correctRate: number;
  wrongCount: number;
  lastReviewed?: string;
}

// 学习目标
interface LearningGoal {
  id: string;
  domain: string;
  targetScore: number;
  currentScore: number;
  deadline: string;
  dailyTarget: number;
}
```

## Technical Notes

- 分析算法：基于 quizStats 计算
- 推荐算法：贪心策略 + 依赖拓扑排序
- 诊断自测：每领域按难度分层抽样
