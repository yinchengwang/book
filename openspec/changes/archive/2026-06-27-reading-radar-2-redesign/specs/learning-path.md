# Learning Path 学习路径

## Overview

学习路径以 Step-by-Step 的方式引导用户系统学习，提供进度追踪和推荐学习顺序。

## Features

### F9.3 路径配置

**描述**：自定义学习路径

**配置项**：
| 配置 | 说明 |
|------|------|
| 添加步骤 | 选择领域加入路径 |
| 排序 | 拖拽调整顺序 |
| 设置前置 | 指定步骤依赖 |
| 设定目标 | 截止日期 + 目标分数 |

**前置依赖**：
```
Step 3: AI 基础理论
  前置条件：Step 1 已完成
  依赖主题：Python 基础、函数、类
```

### F9.5 路径统计

**描述**：路径整体统计

**指标**：
| 指标 | 计算 |
|------|------|
| 总体进度 | 已完成步骤 / 总步骤 |
| 学习时长 | 累计学习时间估算 |
| 预计完成 | 基于当前速度预测 |
| 偏离程度 | 当前进度 vs 计划进度 |

## Data Models

```typescript
// 学习步骤
interface LearningStep {
  id: string;
  order: number;
  domain: string;         // 领域 ID
  title: string;
  description: string;
  priority: 'critical' | 'high' | 'normal';
  duration: string;       // 预估时长
  prerequisites: string[]; // 前置步骤 ID
  status: 'locked' | 'active' | 'completed';
}

// 路径配置
interface LearningPath {
  id: string;
  name: string;
  steps: LearningStep[];
  createdAt: string;
  updatedAt: string;
}
```

## Technical Notes

- 进度计算：基于 quizStats 数据
- 推荐算法：基于正确率和学习频率
- 路径可视化：自定义 SVG 或 Recharts
- 路径数据：可导出为 JSON/CSV
