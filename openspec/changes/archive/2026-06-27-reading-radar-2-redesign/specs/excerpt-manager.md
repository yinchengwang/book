# Excerpt Manager 摘抄管理

## Overview

摘抄管理用于记录读书笔记、摘录金句，支持 Markdown 格式和 Obsidian 双向链接。

## Features

### F8.1 摘抄列表

**描述**：按书籍/标签分类的摘抄列表

**视图切换**：
| 视图 | 说明 |
|------|------|
| 书籍视图 | 按书籍分组 |
| 标签视图 | 按标签分组 |
| 时间视图 | 按时间倒序 |

**筛选条件**：
- 书籍：`select` 下拉选择
- 标签：`tag` 筛选，支持多选
- 年份：`year` 筛选
- 搜索：`keyword` 全文搜索

**列表项**：

### F8.4 Obsidian 双向链接

**描述**：与 Obsidian 笔记的双向链接

**链接格式**：
```markdown
[[笔记名]]           // 单向链接
[[笔记名|显示文本]]   // 带显示文本
![[笔记名]]          // 嵌入笔记内容
```

**链接功能**：
| 功能 | 说明 |
|------|------|
| 创建链接 | 输入 `[[` 触发自动补全 |
| 预览链接 | 悬停显示笔记摘要 |
| 跳转到 Obsidian | 点击打开 Obsidian 应用 |
| 反向链接 | 显示所有引用此笔记的摘抄 |

**Obsidian URI**：
```
obsidian://open?vault=MyVault&file=笔记名
```

## Data Models

```typescript
// 摘抄
interface Excerpt {
  id: string;
  bookId: string;
  quote: string;          // 摘录原文
  thoughts?: string;       // 个人思考
  tags: string[];
  favorite: boolean;
  createdAt: string;
  updatedAt: string;
}

// 书籍
interface Book {
  id: string;
  title: string;
  author: string;
  cover?: string;
  excerptCount: number;
  createdAt: string;
  updatedAt: string;
}

// 标签
interface Tag {
  name: string;
  count: number;
  color?: string;
}
```

## Technical Notes

- Markdown 渲染：`react-markdown` + `remark-gfm`
- 编辑器：基于 `textarea` 或 CodeMirror
- Obsidian 链接：解析 `[[...]]` 语法
- 存储：localStorage + Markdown 文件
- 搜索：全文搜索（低版本）或 IndexedDB 索引
