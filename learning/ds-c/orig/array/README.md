# 数组 (Array)

## 概述

数组是最基本的数据结构，用一段连续的内存空间存储相同类型的元素。

## 基本操作

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 随机访问 | O(1) | 通过索引直接访问 |
| 查找 | O(n) | 遍历查找 |
| 插入 | O(n) | 需要移动后续元素 |
| 删除 | O(n) | 需要移动后续元素 |

## 可视化示例

### 数组结构

```
索引:   [0]    [1]    [2]    [3]    [4]
        ┌──────┬──────┬──────┬──────┬──────┐
数组:   │  1   │  2   │  3   │  4   │  5   │
        └──────┴──────┴──────┴──────┴──────┘
内存:   0x100  0x104  0x108  0x10C  0x110
```

### 插入操作示例

在数组 `[1, 2, 4, 5]` 中插入 `3` 到索引 2：

```mermaid
graph LR
    A["[1, 2, 4, 5]"] --> B["插入 3"]
    B --> C["[1, 2, <span style='color:#9f9'>3</span>, 4, 5]"]
    style C fill:#e8f5e9
    style A fill:#fff3e0
```

步骤说明：
1. 从末尾开始，将 `[4, 5]` 向后移动一位
2. 在索引 2 位置插入 `3`
3. 结果：`[1, 2, 3, 4, 5]`（新增元素用绿色标识）

### 删除操作示例

从数组 `[1, 2, 3, 4, 5]` 中删除索引 2 的元素 `3`：

```mermaid
graph LR
    A["[1, 2, <span style='color:#f96'>3</span>, 4, 5]"] --> B["删除"]
    B --> C["[1, 2, 4, 5]"]
    style A fill:#ffebee
    style C fill:#e8f5e9
```

步骤说明：
1. 标记要删除的元素 `3`
2. 将 `[4, 5]` 向前移动
3. 结果：`[1, 2, 4, 5]`（被删元素用红色标识）
