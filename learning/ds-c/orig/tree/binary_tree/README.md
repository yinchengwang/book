# 二叉树 (Binary Tree)

## 概述

二叉树是每个节点最多有两个子树的树结构，通常子树被称作"左子树"和"右子树"。

## 基本操作

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 遍历（前/中/后序） | O(n) | 访问所有节点 |
| 查找 | O(n) | 最坏情况退化成链表 |
| 插入 | O(n) | 需要维护平衡 |
| 删除 | O(n) | 需要维护平衡 |

## 可视化示例

### 二叉树结构

```
        1
       / \
      2   3
     / \   \
    4   5   6
```

### 遍历顺序对比

```mermaid
graph TD
    subgraph 前序遍历
        A1["根 → 左 → 右"]
        A2["1 → 2 → 4 → 5 → 3 → 6"]

    subgraph 中序遍历
        B1["左 → 根 → 右"]
        B2["4 → 2 → 5 → 1 → 3 → 6"]
    end
    
    subgraph 后序遍历
        C1["左 → 右 → 根"]
```
