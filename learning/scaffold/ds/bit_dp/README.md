# 位运算 DP (Bit DP)

## 概述

本模块演示位运算与状态压缩动态规划：

| 内容 | 作用 | 复杂度 |
|------|------|--------|
| 状态压缩 | 用位掩码表示集合状态 | - |
| TSP DP | 旅行商问题精确解 | O(n^2 * 2^n) |
| 集合覆盖 | 贪心近似算法 | O(m * n) |

## 关键概念

### 状态压缩

- 用整数的二进制位表示集合状态
- `x & (x-1)` 消除最低位的 1
- `x & (-x)` 获取最低位的 1

### TSP DP

- `dp[mask][i]` 表示已访问集合 mask，当前在顶点 i
- 状态转移：添加未访问顶点
- 时间复杂度高，适合 n ≤ 20

### 位运算技巧

```c
x & (-x)     /* 取最低位的 1 */
x & (x-1)    /* 消除最低位的 1 */
(x & (x-1)) == 0  /* 判断 2 的幂 */
```
