# 数据结构学习卡片：数组 (Array)

## 学习目标

1. 理解静态数组与动态数组的区别
2. 掌握数组的随机访问特性（O(1) 时间复杂度）
3. 理解动态数组的扩容策略（2倍扩展）
4. 掌握数组插入与删除操作的时间复杂度（O(n)）

## 核心概念

### 静态数组

- **定义**：编译时确定大小的固定长度数组
- **存储位置**：栈上分配（小型数组）
- **优点**：访问速度快，无需手动管理内存
- **缺点**：大小固定，无法动态调整

### 动态数组

- **定义**：运行时根据需要扩展的数组
- **存储位置**：堆上分配（malloc/calloc）
- **扩容策略**：常用 2 倍扩展（amortized O(1)）
- **实现要点**：
  - `malloc` 初始分配
  - `realloc` 扩容时迁移数据
  - 容量（capacity）与大小（size）的区分

### 时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 随机访问 `arr[i]` | O(1) | 直接计算地址 |
| 尾部插入 `push` | O(1) amortized | 均摊分析 |
| 指定插入 `insert` | O(n) | 需移动元素 |
| 指定删除 `delete` | O(n) | 需移动元素 |
| 扩容 | O(n) | 数据迁移 |
