# 数据结构学习卡片：栈 (Stack)

## 学习目标

1. 理解栈的基本概念：LIFO（后进先出）
2. 掌握顺序栈（数组实现）和链式栈（链表实现）
3. 理解栈在函数调用、表达式求值中的应用
4. 掌握中缀表达式转后缀表达式的算法

## 核心概念

### 栈的基本操作

- **Push**：将元素压入栈顶
- **Pop**：弹出栈顶元素
- **Top**：获取栈顶元素（不出栈）
- **Empty**：检查栈是否为空

### 顺序栈（数组实现）

- **结构**：使用动态数组存储，栈顶指针为数组下标
- **优点**：随机访问 O(1)，缓存友好
- **缺点**：需要扩容操作
- **扩容策略**：常用 2 倍扩展（amortized O(1)）

### 链式栈（链表实现）

- **结构**：栈顶在链表头部（头插头删）
- **优点**：无容量限制，入栈出栈 O(1)
- **缺点**：每个元素需要额外指针存储

### 栈的应用场景

| 应用 | 说明 |
|------|------|
| 函数调用 | 保存返回地址、参数、局部变量 |
| 表达式求值 | 中缀转后缀、后缀求值 |
| 括号匹配 | 验证括号是否配对 |
| 撤销/回溯 | 编辑器的 Undo 操作 |
| 深度优先搜索 | DFS 使用栈保存路径 |

### 时间复杂度

| 操作 | 顺序栈 | 链式栈 |
|------|--------|--------|
| Push | O(1) amortized | O(1) |
| Pop | O(1) | O(1) |
| Top | O(1) | O(1) |
