# AVL 树（AVL Tree）

AVL 树是一种自平衡二叉搜索树，由 G.M. Adelson-Velsky 和 E.M. Landis 在 1962 年提出。
它是计算机科学中最早被发明的自平衡二叉树。

## 核心特性

### 平衡条件

任意节点的左右子树高度差（平衡因子）绝对值不超过 1：

```
平衡因子 = 左子树高度 - 右子树高度
|平衡因子| <= 1
```

### 四种旋转操作

| 类型 | 失衡情况 | 旋转方式 |
|------|---------|---------|
| LL | 左子树的左孩子导致失衡 | 单次右旋 |
| RR | 右子树的右孩子导致失衡 | 单次左旋 |
| LR | 左子树的右孩子导致失衡 | 先左旋后右旋 |
| RL | 右子树的左孩子导致失衡 | 先右旋后左旋 |

### 时间复杂度

| 操作 | 平均 | 最坏 |
|------|------|------|
| 查找 | O(log n) | O(log n) |
| 插入 | O(log n) | O(log n) |
| 删除 | O(log n) | O(log n) |

## 与普通 BST 的区别

- BST 在最坏情况下退化为链表（O(n)）
- AVL 树始终保持平衡，查找效率稳定

## 使用方法

```bash
make        # 编译
make run    # 运行
make clean  # 清理
```
