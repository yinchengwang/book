# 红黑树 (Red-Black Tree)

## 概述

红黑树是一种自平衡二叉搜索树，通过节点颜色标记和旋转操作维护平衡性，保证最坏情况下的查找、插入、删除操作均为 **O(log n)**。

## 五大性质

1. 每个节点非红即黑
2. 根节点为黑
3. 叶节点(NIL)为黑
4. 红节点的孩子必须为黑（无红红冲突）
5. 从任一节点到其每个叶子的路径上，黑高相同

## 核心操作

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 查找 | O(log n) | BST 查找路径 |
| 插入 | O(log n) | BST 插入 + O(1) 旋转修复 |
| 删除 | O(log n) | BST 删除 + O(log n) 旋转修复 |

## 插入修复三情形

- **Case 1**：叔节点为红 -> 父叔同黑，祖父变红，递归处理
- **Case 2**：叔节点为黑且 z 为右孩子 -> 左旋转变为 Case 3
- **Case 3**：叔节点为黑且 z 为左孩子 -> 右旋转变色

## 运行

```bash
make run
```
