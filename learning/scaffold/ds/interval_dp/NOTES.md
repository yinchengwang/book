# 区间 DP (Interval DP)

## 核心思想

区间 DP 对区间 [i, j] 进行状态规划，状态转移考虑区间分割点 k：

```
dp[i][j] = min(dp[i][k] + dp[k+1][j] + cost(i,j))
```

## 经典问题

| 问题 | 复杂度 | 说明 |
|------|--------|------|
| 矩阵链乘 | O(N³) | 括号化顺序 |
| 石子合并 | O(N³) | 环形/线性 |
| 区间调度 | O(N log N) | 贪心近似 |
