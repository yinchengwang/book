# Graph Traversal - 图的遍历

## 概述

本 scaffold 演示图遍历的核心算法，包括 BFS、DFS、连通分量检测和拓扑排序。

## 目录结构

```
graph_traversal/
├── main.c      # 演示程序
├── Makefile    # 编译配置
├── README.md   # 本文件
└── NOTES.md    # 工程对照说明
```

## 编译运行

```bash
make        # 编译
make run    # 编译并运行
make clean  # 清理
```

## 算法实现

| 算法 | 核心数据结构 | 时间复杂度 |
|------|-------------|-----------|
| BFS | 队列 | O(V + E) |
| DFS | 递归/栈 | O(V + E) |
| 连通分量 | DFS 扩展 | O(V + E) |
| 拓扑排序 (Kahn) | 队列 + 入度 | O(V + E) |
| 拓扑排序 (DFS) | 后序逆序 | O(V + E) |

## 示例输出

```
--- 示例 1：无向图 ---
BFS: 0 1 3 2
DFS: 0 1 2 3
连通分量数: 2

--- 示例 2：有向图（课程先修关系） ---
拓扑排序 (Kahn): 0 1 2 3 4
拓扑排序 (DFS): 0 1 2 3 4
```

## 关键概念

- **BFS**：层层扩展，适合求最短路径
- **DFS**：深度优先，适合连通分量、环检测
- **拓扑排序**：DAG 特有，应用于任务调度
