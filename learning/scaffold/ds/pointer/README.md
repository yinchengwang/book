# 数据结构学习卡片：指针 (Pointer)

## 学习目标

1. 理解指针的本质：指针是一个变量，存储另一个变量的地址
2. 掌握指针的基本操作：取地址 `&`、解引用 `*`
3. 理解指针算术运算：`++`, `--`, `+`, `-` 与数组下标的等价关系
4. 掌握函数指针的概念：函数作为一等公民，可以像变量一样传递
5. 理解 const 指针的三种模式及其使用场景

## 核心概念

### 指针基础

| 概念 | 符号 | 说明 |
|------|------|------|
| 取地址 | `&var` | 获取变量的内存地址 |
| 解引用 | `*ptr` | 访问指针所指向的内存位置 |
| 空指针 | `NULL` | 表示指针不指向任何有效地址 |

### 指针算术

```c
int arr[] = {10, 20, 30};
int *p = arr;           // p 指向 arr[0]
*(p + 1);               // 等价于 arr[1]
p++;                    // p 指向 arr[1]
p2 - p;                 // 两个指针的距离（元素个数）
```

### 函数指针

```c
// 函数指针类型定义
typedef int (*Operation)(int, int);

// 直接声明
int (*op)(int, int) = add;

// 使用
op(3, 4);               // 调用 add(3, 4)

// 函数指针数组
Operation ops[] = {add, multiply, max_int};
ops[0](1, 2);           // 调用 add(1, 2)

// 回调函数
int apply(int a, int b, int (*func)(int, int));
```

### const 指针模式

| 模式 | 语法 | 约束 |
|------|------|------|
| 指向 const 的指针 | `const int *p` | 不能通过 p 修改 *p |
| const 指针 | `int * const p` | p 不能指向其他地址 |
| const 指针指向 const | `const int * const p` | 两者都不能修改 |

### 数组与指针的关系

- 数组名在大多数表达式中退化为指向首元素的指针
- `arr[i]` 等价于 `*(arr + i)` 等价于 `*(p + i)`（p = arr）
- `&arr` 是数组的地址，`arr` 是首元素的地址，两者不同

## 时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 指针解引用 `*p` | O(1) | 直接访问内存 |
| 指针算术 `p+n` | O(1) | 计算新地址 |
| 函数指针调用 `func()` | O(1) | 间接调用 |
