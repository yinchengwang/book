# 动态规划 (Dynamic Programming)

## 概述

本模块演示三种经典 DP 问题：

| 问题 | 状态定义 | 递推公式 | 复杂度 |
|------|----------|----------|--------|
| 斐波那契 | dp[i] = 第 i 项值 | dp[i] = dp[i-1] + dp[i-2] | O(n) |
| 0-1 背包 | dp[i][w] = 前 i 物容量 w 的最大价值 | dp[i][w] = max(dp[i-1][w], dp[i-1][w-wi]+vi) | O(nW) |
| 编辑距离 | dp[i][j] = s1 前 i 到 s2 前 j 的距离 | dp[i][j] = min(dp[i-1][j]+1, dp[i][j-1]+1, dp[i-1][j-1]+cost) | O(mn) |

## 关键概念

### DP 适用条件

1. **最优子结构**: 问题的最优解可由子问题的最优解构造
2. **重叠子问题**: 子问题空间小，递归树有大量重复计算
3. **无后效性**: 状态一旦确定，后续决策不受之前路径影响

### DP 实现方式

- **自顶向下 + 记忆化**: 递归 + 缓存，代码直观
- **自底向上**: 拓扑序迭代，空间/时间更优
- **空间优化**: 从二维压缩到一维（如背包问题）
