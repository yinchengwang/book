# BTree Scaffold

## 简介

本 scaffold 实现了 m 叉 BTree 的核心操作，用于学习 BTree 数据结构的原理。

## 主要功能

- **插入**：支持自动节点分裂
- **查找**：递归查找指定键
- **遍历**：中序遍历所有键

## 编译运行

```bash
make        # 编译
make run    # 编译并运行
make clean  # 清理
```

## 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| MAX_KEYS | 3 | 每个节点最多键数 |
| MIN_KEYS | 1 | 每个节点最少键数 |
| MAX_CHILDREN | 4 | 每个节点最多子节点数 |

## 键值

- 阶数 m = 4（每个节点最多 4 个子节点）
- 每个节点最多 3 个键
