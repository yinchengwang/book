# 树形 DP (Tree DP)

## 概述

本模块演示三种树形动态规划问题：

| 问题 | 作用 | 时间复杂度 |
|------|------|-----------|
| 树的直径 | 求最长路径 | O(V) |
| 最大独立集 | 选择最大互不相邻节点 | O(V) |
| 树形背包 | 树上分组背包 | O(V * K^2) |

## 关键概念

### 树的直径

- 两次 DFS：从任意节点出发找最远点 A
- 再从 A 出发找最远点 B，A-B 即为直径

### 最大独立集

- `include[u]` = 选 u 时以 u 为根的最大独立集大小
- `exclude[u]` = 不选 u 时的最大独立集大小
- 状态转移：`include[u] += exclude[child]`

### 树形背包

- 将每棵子树视为一组物品
- 组内互斥（选不同的子树大小）
- 合并时使用 0/1 背包的逆序更新
