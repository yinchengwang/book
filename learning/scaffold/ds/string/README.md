# 数据结构学习卡片：字符串 (String)

## 学习目标

1. 理解 C 字符串的本质（char 数组 + '\0' 结尾）
2. 掌握常用字符串函数的使用：strlen, strcpy, strcat, strcmp
3. 了解字符串操作中的常见陷阱（缓冲区溢出、'\0' 处理）
4. 建立 UTF-8 编码意识，理解中文字符的多字节特性

## 核心概念

### C 字符串的本质

- **定义**：以空字符 '\0' (0x00) 结尾的 char 数组
- **重要**：`strlen()` 返回的长度**不含**结尾的 '\0'
- **存储位置**：字符串字面量通常在只读数据段，字符数组在栈/堆上

### 常用字符串函数

| 函数 | 功能 | 注意事项 |
|------|------|----------|
| `strlen(s)` | 返回字符串长度（不含 '\0'） | O(n)，需遍历 |
| `strcpy(dst, src)` | 复制字符串 | 目标缓冲区必须够大 |
| `strncpy(dst, src, n)` | 至多复制 n 字节 | 不保证加 '\0' |
| `strcat(dst, src)` | 追加字符串 | dst 必须有足够空间 |
| `strcmp(s1, s2)` | 比较字符串 | 字典序，逐字符比较 |

### UTF-8 编码

- ASCII 字符（0-127）：1 字节
- 中文/日文/韩文：通常 3 字节（UTF-8 编码范围 E0-EF）
- `strlen()` 返回的是**字节数**，不是字符数
- 处理多字节字符需使用专门的 Unicode 库（如 iconv、ICU）

### 时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| `strlen()` | O(n) | 需遍历找到 '\0' |
| `strcpy()` / `strcat()` | O(n) | 复制/追加所有字符 |
| `strcmp()` | O(n) | 最坏情况遍历全部 |
