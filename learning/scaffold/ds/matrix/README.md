# 矩阵运算 (Matrix Operations)

## 概述

本模块演示三大矩阵运算：矩阵乘法、矩阵快速幂、稀疏矩阵压缩。

| 内容 | 时间复杂度 | 应用场景 |
|------|-----------|----------|
| 矩阵乘法 | O(n³) | 线性代数、图论 |
| 矩阵快速幂 | O(n³ log k) | 斐波那契、路径计数 |
| 稀疏矩阵 CSR | O(nnz) | 稀疏图、科学计算 |

## 关键概念

### 矩阵乘法

三层循环计算 C[i][j] = Σ A[i][k] × B[k][j]：
- 标准算法：直接三层循环 O(n³)
- 分块优化：提高 CPU 缓存命中率

### 矩阵快速幂

利用二分幂思想加速矩阵幂运算：
- A^n = A^(n/2) × A^(n/2) （n 为偶数）
- A^n = A × A^(n-1) （n 为奇数）
- 斐波那契：`[F(n), F(n-1)] = A^(n-1) × [1, 0]`

### CSR 稀疏矩阵格式

Compressed Sparse Row (CSR) 三数组结构：
- `values`: 非零值依次存储
- `col_indices`: 对应的列索引
- `row_ptr`: 每行起始位置
- 适合按行遍历的稀疏矩阵
