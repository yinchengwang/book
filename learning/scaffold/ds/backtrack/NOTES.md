# 回溯算法 (Backtracking)

## 核心思想

回溯是深度优先搜索的扩展，通过"选择-递归-撤销"模式穷举所有可能解。

## 经典问题

| 问题 | 复杂度 | 应用 |
|------|--------|------|
| N 皇后 | O(N!) | 约束满足 |
| 全排列 | O(N!) | 排列生成 |
| 子集生成 | O(2^N) | 组合枚举 |

## 回溯三步骤

1. **选择 (Choose)**: 做出一个选择
2. **探索 (Explore)**: 递归搜索
3. **撤销 (Unchoose)**: 回退状态

## 工程对照

### 约束满足问题（CSP）
- 调度排班、数独求解
- 编译器寄存器分配

### 组合优化
- 旅行商问题（TSP）暴力解
- 图着色问题

## 运行

```bash
gcc -std=c11 -Wall -Wextra -O2 -o backtrack_demo main.c && ./backtrack_demo
```
