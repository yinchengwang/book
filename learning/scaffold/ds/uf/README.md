# 并查集（Union-Find）Scaffold

## 概述

本目录实现了一个简洁的并查集数据结构演示，包含两大核心优化：

| 优化 | 说明 |
|------|------|
| 路径压缩（Path Compression） | find 递归返回时将路径上所有节点直接指向根，使树高趋近常数 |
| 按秩合并（Union by Rank） | 将小树合并到大树下，保持整体平衡，抑制树高增长 |

两者结合后，并查集的每次操作均摊时间复杂度约为 **O(α(n))**，其中 α 是 Ackermann 函数的反函数，在实际应用中可视为常数（n < 10^80 时 α(n) ≤ 4）。

```bash
make        # 编译
make run    # 编译并运行
make clean  # 清理
```

## 核心接口

```c
uf_t *uf_create(int n);          /* 创建并查集 */
void  uf_destroy(uf_t *uf);      /* 销毁 */
int   uf_find(uf_t *uf, int x);  /* 查找根节点（路径压缩） */
void  uf_union(uf_t *uf, int x, int y);  /* 合并（按秩） */
int   uf_connected(uf_t *uf, int x, int y);  /* 判断连通性 */
```

## 典型应用场景

- **连通性判断**：图论中判断两点是否连通
- **集合划分**：动态合并与查询元素的分组
- **最小生成树**：Kruskal 算法中检测环
- **等价类处理**：网络延迟合并、区块链分叉检测等

## 演示输出

运行后将输出：
1. 初始状态（每个元素独立）
2. 合并操作序列
3. 连通性查询结果
4. 10x10 连通性矩阵
5. 路径压缩后的节点状态变化
