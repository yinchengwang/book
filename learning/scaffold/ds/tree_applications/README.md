## 演示内容

### 1. 表达式树 (Expression Tree)

用于解析和计算数学表达式，支持：
- 前缀表达式构建树
- 递归求值
- 支持变量（a-z）

**示例**：`(a + b) * c` 的树结构：
```
       *
      / \
     +   c
    / \
   a   b
```

### 2. Huffman树 (Huffman Tree)

用于数据压缩，生成最优前缀编码：
- 基于字符频率构建最优二叉树
- 生成变长编码，高频字符用短码
- 计算压缩率

**示例**：
- 输入：`aabbbcccc`
- 输出：`a: 10, b: 11, c: 0`（压缩率 ~56%）

### 3. 树状数组 (Fenwick Tree)

用于高效的前缀和查询：
- 单点更新：O(log n)
- 前缀和查询：O(log n)
- 区间和查询：O(log n)

**示例**：
- 数组：`[1, 3, 5, 7, 9, 11]`
- 前缀和 `sum[0..3] = 16`
- 区间和 `sum[2..4] = 21`

## 学习要点

1. **树的三种遍历**：前序、中序、后序
2. **递归思维**：树的天然递归结构
3. **空间换时间**：Huffman编码 vs 定长编码
4. **树状数组**：利用二进制位运算优化
