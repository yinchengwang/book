# 数据结构学习卡片：高级排序算法 (Advanced Sorting)

## 学习目标

1. 理解快速排序的分区思想与常见优化策略
2. 掌握归并排序的分治实现与稳定性保证
3. 理解堆排序的建堆与下沉调整过程
4. 掌握各排序算法的复杂度分析与适用场景

## 核心概念

### 快速排序 (Quick Sort)

- **分区思想**：选取 pivot，将数组分为 <= pivot 和 > pivot 两部分
- **优化策略**：
  - 三数取中选择 pivot（避免有序数组的最差情况 O(n^2)）
  - 小数组用插入排序（减少递归栈开销）
  - 三路划分（处理大量重复元素）
- **时间复杂度**：平均 O(n log n)，最差 O(n^2)

### 归并排序 (Merge Sort)

- **分治思想**：先递归排序子数组，再合并两个有序子数组
- **特点**：
  - 稳定排序
  - 需要 O(n) 额外空间
  - 适合链表排序（只需 O(1) 空间）
- **时间复杂度**：总是 O(n log n)

### 堆排序 (Heap Sort)

- **堆结构**：完全二叉树，可用数组实现
- **步骤**：建堆（从最后一个非叶子节点下沉）+ 逐个取出堆顶
- **特点**：
  - 原地排序，O(1) 额外空间
  - 不稳定排序
  - 不受输入数据分布影响
- **时间复杂度**：总是 O(n log n)

## 复杂度对比

| 排序算法 | 最优 | 平均 | 最差 | 空间 | 稳定性 |
|---------|------|------|------|------|--------|
| 快速排序 | O(n log n) | O(n log n) | O(n^2) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 |
