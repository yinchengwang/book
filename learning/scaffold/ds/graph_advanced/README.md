# 图高级算法 (Graph Advanced)

## 概述

本模块演示两种图高级算法：

| 算法 | 作用 | 时间复杂度 |
|------|------|-----------|
| Tarjan SCC | 求强连通分量 | O(V+E) |
| 欧拉回路检测 | 判断是否存在欧拉路径 | O(V+E) |

## 关键概念

### Tarjan 强连通分量

- 使用 `dfn[]` (时间戳) 和 `low[]` (能追溯的最小 dfn)
- `low[u] == dfn[u]` 时弹出栈得到一个 SCC
- 核心思想：DFS 树中的最低祖先

### 欧拉回路条件

**无向图：**
- 欧拉回路：所有顶点度数为偶数
- 欧拉通路：恰好 2 个奇度顶点

**有向图：**
- 欧拉回路：每个顶点出入度相等
- 欧拉通路：恰好 1 个起点(出>入)、1 个终点(入>出)
