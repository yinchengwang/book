# 数据结构学习卡片：位运算 (Bitwise Operations)

## 学习目标

1. 掌握基础位运算符：AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), 左移 (`<<`), 右移 (`>>`)
2. 理解位操作技巧：设置位、清除位、翻转位、检查位、提取位域
3. 理解位图（Bitmap）的原理及其在集合成员检测中的应用
4. 理解布隆过滤器（Bloom Filter）原理及其概率特性
5. 了解位运算在工程代码中的实际应用场景

## 核心概念

### 基础位运算

| 运算符 | 名称 | 规则 | 典型应用 |
|--------|------|------|----------|
| `&` | 按位与 | 两位都为1时结果为1 | 提取特定位、清除特定位 |
| `\|` | 按位或 | 任一位为1时结果为1 | 设置特定位 |
| `^` | 按位异或 | 两位不同时结果为1 | 加密、两数交换、找唯一数 |
| `~` | 按位取反 | 0变1, 1变0 | 位模式反转 |
| `<<` | 左移 | 高位丢弃，低位补0 | 乘以2^n |
| `>>` | 右移 | 低位丢弃，高位补0/符号位 | 除以2^n |

### 位操作技巧

```c
// 设置第 n 位为1
value | (1U << n)

// 清除第 n 位（置0）
value & ~(1U << n)

// 翻转第 n 位
value ^ (1U << n)

// 检查第 n 位是否为1
(value >> n) & 1U

// 提取连续位域 [start, start+len-1]
(value >> start) & ((1U << len) - 1)
```

### 位图（Bitmap）

- **原理**：用每一位表示一个元素是否存在（1=存在，0=不存在）
- **空间效率**：1个元素仅占1位，vs 1个bool占1字节（8倍压缩）
- **时间复杂度**：O(1) 查找、插入、删除
- **应用**：数据库索引、集合去重、内存页分配位图、IP地址检测

### 布隆过滤器（Bloom Filter）

- **原理**：使用多个哈希函数将元素映射到位图的多个位
- **特性**：
  - **无假阴性**：不存在的元素一定报告"不存在"
  - **有假阳性**：存在的元素可能报告"存在"（但概率可控）
- **应用**：缓存穿透防护、网页去重、垃圾邮件过滤、黑名单
- **空间计算**：`m = -n * ln(p) / (ln(2)^2)`，其中 n=元素数，p=假阳性率

## 时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 单比特操作 | O(1) | 设置、清除、翻转、检查 |
| 位图查询 | O(1) | 直接寻址 |
| 位图插入 | O(1) | 设置对应位 |
| 布隆过滤器查询 | O(k) | k=哈希函数数量 |
| 布隆过滤器插入 | O(k) | 设置k个位 |
