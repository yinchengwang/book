# 图的表示方法 (Graph Representation)

## 概述

本模块演示两种基本图表示法：

| 表示法 | 空间复杂度 | 边查询 | 遍历邻居 | 适用场景 |
|--------|-----------|--------|----------|----------|
| 邻接矩阵 | O(V^2) | O(1) | O(V) | 稠密图 |
| 邻接表 | O(V+E) | O(deg) | O(deg) | 稀疏图 |

## 关键概念

### 邻接矩阵 (Adjacency Matrix)

- 使用 N×N 二维数组存储边信息
- `matrix[i][j] = 1` 或权重表示边 (i,j) 存在
- 无向图矩阵对称

### 邻接表 (Adjacency List)

- 每个顶点维护一条出边链表
- 边节点包含: 目标顶点、权重、下一指针
- 更节省稀疏图空间
