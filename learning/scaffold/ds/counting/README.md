# 计数原理 (Counting Principles)

## 概述

本模块演示三大计数技术：排列组合、容斥原理、卡特兰数。

| 内容 | 核心公式 | 应用场景 |
|------|----------|----------|
| 排列组合 | P(n,r), C(n,r) | 选人、路径计数 |
| 容斥原理 | \|A∪B\| = \|A\|+\|B\|-\|A∩B\| | 集合交集计数 |
| 卡特兰数 | C_n = C(2n,n)/(n+1) | 合法括号、二叉树 |

## 关键概念

### 排列 (Permutation)

从 n 个不同元素中取出 r 个按顺序排列：`P(n,r) = n!/(n-r)!`

### 组合 (Combination)

从 n 个不同元素中取出 r 个不计较顺序：`C(n,r) = n!/(r!(n-r)!)`

### 容斥原理 (Inclusion-Exclusion)

避免重复计数的核心技巧：
- 两集合：|A∪B| = |A| + |B| - |A∩B|
- 三集合：|A∪B∪C| = Σ|A| - Σ|A∩B| + |A∩B∩C|

### 卡特兰数 (Catalan Number)

合法路径/括号匹配/二叉树计数的通用数列：
- C_0 = C_1 = 1
- C_n = C(2n,n) - C(2n,n+1)
- C_n = C_n-1 × 2(2n-1)/(n+1)
