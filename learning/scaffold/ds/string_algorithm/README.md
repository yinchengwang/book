# 字符串算法 (String Algorithm)

## 概述

本模块演示两种核心字符串算法：

| 算法 | 作用 | 时间复杂度 |
|------|------|-----------|
| KMP | 线性模式匹配 | O(n+m) |
| Trie | 前缀树查询 | O(m) |

## 关键概念

### KMP 算法

- 预处理模式串计算 `next[]` 数组
- 匹配失败时利用已匹配信息，避免回溯主串指针
- 核心思想：最长公共前后缀

### Trie 树

- 每个节点存储 26 个子指针（字母表大小）
- 词尾标记区分前缀和完整单词
- 空间换时间，适合大量短字符串查询

### BM 算法（简介）

- Boyer-Moore，从模式串末尾向左匹配
- 坏字符规则 + 好后缀规则
- 实际应用中通常比 KMP 快
