# 概率与随机 (Probability and Randomness)

## 概述

本模块演示三大随机算法基础：线性同余生成器、蒙特卡洛方法、概率 DP。

| 内容 | 核心公式 | 应用场景 |
|------|----------|----------|
| LCG | X_{n+1} = (a·X_n + c) mod m | 伪随机数生成 |
| 蒙特卡洛 | 采样 → 计数 → 估算 | PI/积分/期权定价 |
| 概率 DP | dp[i] = Σ p·dp[j] | 路径概率/期望 |

## 关键概念

### 线性同余生成器 (LCG)

最简单的伪随机数生成器：
- 状态更新：`X_{n+1} = (a·X_n + c) mod m`
- 参数选择影响周期和统计特性
- glibc 使用：a=1103515245, c=12345, m=2^31

### 蒙特卡洛方法

通过随机采样估算确定性问题的数值方法：
- PI：`圆内点数 / 总点数 ≈ π/4`
- 积分：`曲线下方点数 / 总点数 ≈ ∫f(x)dx`
- 收敛速度：O(1/√n)

### 概率 DP

在状态转移中引入概率的动态规划：
- 状态：到达每个位置的概率
- 转移：`dp[i] = Σ (dp[j] × p_{j→i})`
- 应用：路径规划、游戏概率、机器学习
