# 贪心算法 (Greedy Algorithm)

## 概述

本模块演示三种经典贪心问题：

| 问题 | 贪心策略 | 正确性条件 | 复杂度 |
|------|----------|------------|--------|
| 活动选择 | 选结束最早的活动 | 区间可重叠 | O(n log n) |
| Huffman 编码 | 合并频率最小的两节点 | 前缀无二义性 | O(n log n) |
| 分数背包 | 取单位价值最高的物品 | 可分割 | O(n log n) |

## 关键概念

### 贪心算法适用条件

1. **贪心选择性质**: 局部最优选择能导致全局最优
2. **最优子结构**: 问题最优解包含子问题最优解

### 贪心 vs 动态规划

| 问题 | 适用算法 | 原因 |
|------|----------|------|
| 分数背包 | 贪心 | 可分割，局部最优 = 全局最优 |
| 0-1 背包 | 动态规划 | 不可分割，需权衡取舍 |
| Huffman 编码 | 贪心 | 频率合并天然满足贪心性质 |
