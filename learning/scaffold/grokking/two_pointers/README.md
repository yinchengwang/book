# 双指针技巧

## 简介

双指针是一种在数组或链表中广泛使用的算法技巧，通过维护两个指针在数据结构上有策略地移动，将 O(n^2) 的暴力解法优化到 O(n)。本卡片演示了三种经典的双指针应用场景：

- **三数之和**：排序 + 双指针，在一维数组中找出所有和为 0 的三元组
- **容器装水（接雨水）**：左右指针根据高度差计算可接雨水量
- **环形链表入口**：快慢指针检测环并定位入口节点

## 运行方法

```bash
# 编译
make

# 运行
make run

# 或直接编译运行
gcc -std=c11 -Wall -Wextra main.c -o two_pointers && ./two_pointers

# 清理
make clean
```
