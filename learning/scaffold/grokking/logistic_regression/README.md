# 逻辑回归 (logistic_regression)

## 简介

逻辑回归演示 —— 二分类问题、Sigmoid 激活函数、交叉熵损失函数与决策边界。纯 numpy 实现，无外部 ML 框架依赖。

## 目录结构

```
logistic_regression/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/logistic_regression
make run
```
