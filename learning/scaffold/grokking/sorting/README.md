# 排序算法实战

## 简介

排序是算法的基础，本 scaffold 涵盖三种实用的排序算法变体：

- **合并区间**：区间合并问题，实际应用如会议室安排、日程冲突检测
- **快速选择**：找第 K 大/小元素，平均时间复杂度 O(n)，优于完全排序的 O(n log n)
- **桶排序**：分布式计数排序，适合范围已知、数据分布均匀的场景

## 运行方法

```bash
# 编译并运行
make run

# 或手动编译运行
gcc -std=c11 -Wall -Wextra -o sorting main.c
./sorting

# 清理
make clean
```

## 代码结构

```
main.c
├── 合并区间
│   ├── Interval 结构
│   ├── compare_interval()       # 排序比较函数
│   └── merge_intervals()        # 区间合并算法
├── 快速选择
│   ├── partition()              # 分区函数（荷兰国旗）
│   └── quick_select()           # 找第 K 大
└── 桶排序
    └── bucket_sort()            # 分布式排序
```

## 算法复杂度

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|-----------|--------|
| 合并区间 | O(n log n) | O(n) | 稳定（排序后合并） |
| 快速选择 | O(n) 平均 | O(1) | 不稳定 |
| 桶排序 | O(n + k) | O(n + k) | 稳定 |
