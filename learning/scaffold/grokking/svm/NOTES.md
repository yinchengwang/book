# 支持向量机 · 学习笔记

## 核心概念

- **最大间隔分类**：SVM 寻找最大化分类间隔（margin）的超平面，间隔由支持向量决定
- **核技巧**：通过核函数将数据隐式映射到高维空间，使线性不可分变为可分
- **软间隔**：引入松弛变量允许部分样本误分类，C 参数控制惩罚强度
- **支持向量**：只有边界上的少数样本决定分类面，其他样本不影响模型

## 工程对照

SVM 的最大间隔思想在本仓库的索引模块中有对应概念：HNSW 索引的参数 ef_construction 和 M 控制着图构建的质量——更大的 ef 和 M 意味更密集的图（类似 SVM 中的软间隔 C 参数），找到的邻居更接近真实最近邻，但也消耗更多内存和构建时间。此外，核技巧的思想在本仓库的 RAG 系统中有所体现：RAG 使用 embedding 模型将文本映射到向量空间（隐式的高维特征空间），然后在向量空间中进行相似性搜索——这正是核函数要解决的问题（映射到高维）。本仓库的 mm_storage.h 中向量引擎使用余弦距离或欧式距离做相似度度量，不同的距离函数相当于不同的核函数选择。
