# 自然语言处理 · 学习笔记

## 核心概念

- **词嵌入**：将词映射到稠密向量空间，语义相近的词向量距离更近
- **TextCNN**：在嵌入序列上应用 1D 卷积，提取 Ngram 特征，Max-over-time Pooling 获取关键特征
- **情感分析**：判断文本的情感倾向（正面/负面），可使用词典方法或机器学习方法
- **语义偏移**：词向量的算术运算可捕捉语义关系（king - man + woman ≈ queen）

## 工程对照

NLP 的文本表示思想在本仓库的索引模块中有直接对应：Full-text Search（全文索引）将文本分词后构建倒排索引——每个词项对应一个倒排列表（类似词嵌入的词汇表查找）。而向量检索（如 hnsw/ivf）则更进一步，将文本的整体语义编码为固定维度的向量，支持语义相似度搜索（接近词嵌入的类比推理）。本仓库的 RAG 系统结合了这两种思想：先用 BM25 做关键词检索（类似词典查找），再用向量检索做语义匹配（类似嵌入匹配），两者融合得到更优结果。这种混合检索架构"词典+语义"的双通路设计，与 NLP 中"基于规则（词典）→基于统计（词嵌入）→基于深度语义"的演进路线一致。本仓库的 PQ 量化可视为一种"词嵌入压缩"——将浮点数向量量化到码本索引，大幅降低存储成本。
