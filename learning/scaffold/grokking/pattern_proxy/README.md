# 代理模式 (Proxy Pattern)

## 简介

代理模式为另一个对象提供替身或占位符以控制访问。本示例展示了
四种代理变体: 虚代理、保护代理、缓存代理、远程代理。

## 目录结构

```
pattern_proxy/
├── main.py      # 代理模式演示代码 (4 种代理)
├── Makefile     # 构建和运行配置
├── README.md    # 本文件
└── NOTES.md     # 学习笔记
```

## 运行方式

```bash
make run
# 或直接运行:
python3 main.py
```

## 示例要点

1. 虚代理: 大文档延迟加载
2. 保护代理: 基于角色的访问控制
3. 缓存代理: TTL 缓存 + 并发安全 + 手动失效
4. 远程代理: 模拟天气服务 RPC 调用
