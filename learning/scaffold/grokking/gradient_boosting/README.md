# 梯度提升 (Gradient Boosting)

## 简介

GBDT 梯度提升树演示 —— 残差逐步拟合、学习率正则化（shrinkage）、加法模型。纯 numpy 实现，无外部 ML 框架依赖。

## 目录结构

```
gradient_boosting/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/gradient_boosting
make run
```
