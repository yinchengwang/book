# 树与图

## 简介

本练习覆盖二叉树的核心算法技巧，包括 BFS/DFS 遍历、对称性判断和路径搜索。这些是面试和工程中处理树结构数据的基础。

## 算法列表

| 算法 | 难度 | 关键词 |
|------|------|--------|
| 二叉树层序遍历（BFS） | 基础 | 队列、层级 |
| 对称树判断 | 中等 | 递归、镜像 |
| 路径总和 | 中等 | DFS、回溯 |

## 运行方法

```bash
# 编译
gcc -std=c11 -Wall -Wextra -O2 -o trees_graphs main.c

# 运行
./trees_graphs

# 或使用 Makefile
make
make run
make clean
```
