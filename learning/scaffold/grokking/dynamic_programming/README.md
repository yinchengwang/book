# 动态规划

动态规划（DP）是算法面试中的高频考点，核心思想是**将大问题拆解为重叠子问题，通过递推求解**。本卡片演示了三种 DP 经典模式：

1. **爬楼梯（基础 DP）** —— 一维递推，目标是找出 DP 状态定义与递推公式
2. **编辑距离（字符串 DP）** —— 二维 DP 表，处理字符串间的插入/删除/替换操作，是 NLP 中文文本纠错的底层算法
3. **股票买卖最佳时机（状态机 DP）** —— 用有限状态机控制交易次数，把"最多两笔交易"约束编码进 DP 状态
