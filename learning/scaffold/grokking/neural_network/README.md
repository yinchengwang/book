# 神经网络 (Neural Network)

## 简介

MLP 多层感知机演示 —— 激活函数（Sigmoid/Tanh/ReLU）、反向传播算法、XOR 问题求解。纯 numpy 实现，无外部深度学习框架依赖。

## 目录结构

```
neural_network/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/neural_network
make run
```
