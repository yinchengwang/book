# 神经网络 · 学习笔记

## 核心概念

- **MLP**：多层感知机，包含输入层、隐藏层、输出层，各层全连接
- **激活函数**：引入非线性，使网络能拟合复杂函数（Sigmoid/Tanh/ReLU，各有优劣）
- **反向传播**：链式法则计算损失对各层参数的梯度，从输出层逐层向前传播
- **前向传播**：输入从输入层逐层计算到输出层，得到预测结果

## 工程对照

神经网络的反向传播算法与本仓库 db 查询优化器的代价估计有异曲同工之处：查询优化器通过代价模型（类似神经网络的损失函数）评估不同执行计划的代价，然后反向调整参数（类似反向传播的梯度下降）寻找最优执行计划。本仓库 `sql_exec.c` 中的执行计划生成逐层分解 SQL 语句（类似前向传播），而代价估算（基于统计信息和索引代价）反向影响执行策略的选择。此外，本仓库的 PQ（Product Quantization）向量量化可以类比为神经网络的嵌入层——将高维向量压缩到低维码本空间，形成更紧凑的表示。MLP 的通用逼近定理在实践中意味着足够深的网络能拟合任何函数，这与本仓库索引模块中多层 B+Tree 或 HNSW 多层图的层次化表达哲学一致。
