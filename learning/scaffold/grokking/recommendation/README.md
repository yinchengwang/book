# 推荐系统 (Recommendation)

## 简介

推荐系统演示 —— User-Based 协同过滤、Item-Based 协同过滤、矩阵分解（SVD 类）、推荐评估指标（Precision/Recall/F1）。纯 numpy 实现，无外部 ML 框架依赖。

## 目录结构

```
recommendation/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/recommendation
make run
```
