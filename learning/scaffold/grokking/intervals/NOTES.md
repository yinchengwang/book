# 区间处理 — 工程对照笔记

## 现实应用

1. **日历调度**：Google Calendar / Outlook 中检测会议冲突、合并空闲时段（忙/闲合并≈区间合并，会议室≈区间染色）。扫描线 O(nlogn) 是最优解。
2. **资源分配**：云计算实例调度——CPU/内存占用区间合并后可统计峰值资源使用量。Kubernetes 调度器中的 pod 亲和/反亲和约束也可以用区间表示。
3. **网络数据包**：防火墙规则中 IP 段/端口段的合并（iptables 规则去重叠）。区间树（Interval Tree）支持动态查询。

## 关键思路

- 排序是预处理核心——将无序重叠转化为有序扫描。
- 扫描线（Sweep Line）思想：将区间的起止拆分为事件，消除二维区间重叠判断，降维为一维差分计数。
- 三种情况的统一处理：完全不重叠、完全包含、部分重叠。
