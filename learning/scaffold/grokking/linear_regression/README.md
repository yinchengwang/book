# 线性回归 (linear_regression)

## 简介

线性回归演示 —— 一元线性回归、多元线性回归、梯度下降优化算法与回归评估指标。纯 numpy 实现，无外部 ML 框架依赖。

## 目录结构

```
linear_regression/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/linear_regression
make run
```
