# 微服务架构

## 简介

系统设计中微服务架构的核心概念，包括服务拆分、API 网关和服务发现的实现与对比。

## 目录结构

```
system-microservices/
├── main.py    # 微服务架构演示代码
├── Makefile   # 运行配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/system-microservices
make run          # 运行演示
```

## 涵盖内容

- 服务拆分: 单体 vs 微服务对比
- API 网关: 路由/聚合/限流
- 服务发现: 注册中心与健康检查
- 微服务治理关键模式
