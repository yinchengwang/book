# 堆

## 简介

堆是一种特殊的完全二叉树，分为最大堆和最小堆。本练习覆盖堆的三个经典应用场景，是面试和工程中的高频考点。

## 算法列表

| 算法 | 难度 | 关键词 |
|------|------|--------|
| Top-K 问题 | 基础 | 最小堆、规模控制 |
| 动态中位数 | 中等 | 两个堆、平衡 |
| K 路合并 | 中等 | 多堆、外部排序 |

## 运行方法

```bash
# 编译
gcc -std=c11 -Wall -Wextra -O2 -o heap main.c

# 运行
./heap

# 或使用 Makefile
make
make run
make clean
```
