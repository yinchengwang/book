# 搜索技巧

二分查找及其变体是算法面试中的基础考点。本卡片演示了以下几种实用场景：

1. **标准二分查找** —— 在有序数组中快速定位目标值，时间复杂度 O(log n)
2. **旋转排序数组搜索** —— 数组在某个未知点旋转后，利用二分思想仍可在 O(log n) 内完成搜索
3. **左右边界搜索** —— 当数组存在重复元素时，快速找到目标值的首次与末次出现位置

## 运行方法

```bash
cd learning/scaffold/grokking/searching/
make        # 编译
make run    # 运行
make clean  # 清理
```
