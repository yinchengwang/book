# 递归思想

## 简介

递归是一种通过函数调用自身来解决问题的编程范式。递归的核心思想是将大问题分解为相似的子问题，直到分解到可以直接解决的最小问题（base case）。

本 scaffold 涵盖：
- 阶乘的递归实现与记忆化优化
- 斐波那契数列的多种解法对比（递归 vs 记忆化 vs 迭代）
- 回溯模板的子集生成实战

## 运行方法

```bash
# 编译并运行
make run

# 或手动编译运行
gcc -std=c11 -Wall -Wextra -o recursion main.c
./recursion

# 清理
make clean
```

## 代码结构

```
main.c
├── 阶乘实现
│   ├── factorial_recursive()     # 普通递归
│   └── factorial_memoized()      # 记忆化递归
├── 斐波那契数列
│   ├── fib_recursive()          # 普通递归（指数级）
│   ├── fib_memoized()           # 记忆化递归
│   └── fib_iterative()          # 迭代版本
└── 回溯模板
    └── backtrack_subsets()      # 子集生成
```
