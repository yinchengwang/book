# 机器学习基础 (ml_fundamentals)

## 简介

机器学习核心概念演示 —— 监督学习 vs 无监督学习、训练/测试集划分、过拟合与欠拟合。纯 Python + numpy 实现，无外部 ML 框架依赖。

## 目录结构

```
ml_fundamentals/
├── main.py    # 演示代码
├── Makefile   # 构建配置
├── README.md  # 本文件
└── NOTES.md   # 学习笔记
```

## 运行方法

```bash
cd learning/scaffold/grokking/ml_fundamentals
make run
```
