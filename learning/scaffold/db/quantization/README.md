# 量化技术

> 演示 SQ / OPQ / AVQ 三种量化技术的训练、编码、解码流程

## 概念

| 量化类型 | 说明 |
|---------|------|
| SQ (标量量化) | 将每个浮点数独立映射到离散值 |
| OPQ (优化乘积量化) | 分段子空间 + 码本编码 |
| AVQ (自适应向量量化) | 根据数据分布自适应调整量化精度 |

## 编译运行

```bash
make
./test
```

## 输出示例

```
【1】标量量化 (SQ)
[SQ] scale=0.0039, offset=-1.5000
原始 [0.50, -0.30, ...]
编码: 128 77 192 ...

【2】优化乘积量化 (OPQ)
编码: [子空间0: 码字5] [子空间1: 码字3]

【3】自适应向量量化 (AVQ)
- 高频区域: 精细量化
- 低频区域: 粗糙量化
```
