# Redis RESP 协议

## 概述

演示 Redis Serialization Protocol (RESP)：
- **类型编码**：5 种数据类型
- **命令解析**：状态机解析
- **管道化**：批量命令优化

## 编译运行

```bash
make
./redis_resp
# 或
make run
```

## 输出示例

```
========================================
   Redis RESP 协议演示
========================================

[resp] ===== RESP 协议类型 =====
[resp] 简单字符串 (+):
[resp]   +OK\r\n
...
```

## RESP 类型

| 前缀 | 类型 | 用途 |
|------|------|------|
| `+` | 简单字符串 | 成功响应 |
| `-` | 错误 | 错误信息 |
| `:` | 整数 | 数值响应 |
| `$` | Bulk String | 二进制安全字符串 |
| `*` | 数组 | 命令/批量响应 |
