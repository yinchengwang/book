# scalar_quant - 标量量化（Scalar Quantization）

## 核心概念

- **标量量化**：对向量的每个维度独立进行量化
- **缩放因子（scale）**：`scale = (max - min) / (2^bits - 1)`
- **零点偏移（zero_point）**：处理非对称数据分布
- **溢出处理**：饱和截断到量化范围边界

## 量化类型

| 类型 | 比特数 | 离散值数量 | 有符号 |
|------|--------|-----------|--------|
| int8 | 8 | 256 | 是 (-128~127) |
| uint8 | 8 | 256 | 否 (0~255) |
| float16 | 16 | ~65536 | 是 |

## 量化公式

```
量化: q = round(val / scale + zero_point)
反量化: val' = (q - zero_point) * scale
```
