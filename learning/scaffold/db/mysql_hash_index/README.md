# 哈希索引与自适应哈希

## 概述
演示哈希索引的原理：O(1) 点查、线性哈希、可扩展哈希、冲突解决。

## 编译运行
```bash
make && ./test.exe
```

## 核心概念
- **哈希索引 O(1) 点查**：通过哈希函数直接定位桶，避免逐行扫描
- **线性哈希（Linear Hashing）**：按顺序分裂桶，分裂指针渐进前进，适合内存敏感场景
- **可扩展哈希（Extendible Hashing）**：局部深度 + 全局深度，桶分裂时可能翻倍目录
- **冲突解决：链地址法**：同一桶内用链表存储冲突键值对
- **哈希索引不支持范围查询**：无法利用哈希值进行有序遍历

## 行为差异
| 特性 | 线性哈希 | 可扩展哈希 |
|------|----------|------------|
| 扩容方式 | 渐进式，按分裂指针顺序 | 瞬间翻倍目录 |
| 空间开销 | 较低 | 可能产生目录冗余 |
| 查找效率 | 可能需要多次哈希 | 一次哈希定位 |
