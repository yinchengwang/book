# diskann - DiskANN (Vamana) 磁盘图索引

## 概述

DiskANN 是微软研究院提出的基于磁盘的向量近似最近邻搜索方案，
核心算法为 **Vamana 图**（一种鲁棒的近邻图索引）。

## 核心组件

### Vamana 图
- **概念**：一种有向近似 KNN 图，从任意节点出发可快速到达最近邻
- **特点**：出度有界（Bounded out-degree），适合磁盘存储
- **构建**：贪心搜索 + 邻居剪枝，保证图连通性的同时控制度上界

### PQ 量化
- **Product Quantization**：将高维向量分段，每段独立做 k-means 聚类
- **编码**：每段用 1 字节（256 簇）表示，128 维向量压缩到 16 字节
- **查表距离**：`dist(x, y) ≈ Σ LUT[x_code[m]][y_code[m]]`

### 混合搜索策略
| 层级 | 存储 | 搜索方式 |
|------|------|----------|
| 索引图 | 内存 | Beam Search 遍历 |
| PQ 码 | 内存 | 快速近似距离过滤 |
| 原始向量 | SSD | 按需读取，精排 |

### Beam Search
从 medoid（入口点）出发，使用优先队列贪心扩展候选集，
遍历 L_SEARCH 个节点后返回 Top-R 结果。

## 编译运行

```bash
make && ./test
```

## 关键参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| L_BUILD | 构建时候选集大小 | 45-100 |
| R | 最大出度（剪枝目标） | 16-64 |
| L_SEARCH | 搜索时遍历上限 | 32-64 |
| PQ_BYTES | PQ 分段数 | 16 (128/16) |
