# numpy - NumPy 科学计算

## 概述

本卡演示 NumPy 核心功能：ndarray、向量化运算、矩阵操作。

## 核心概念

| 概念 | 说明 |
|------|------|
| ndarray | N 维数组，NumPy 的核心数据结构 |
| broadcasting | 广播机制，自动扩展形状匹配 |
| vectorization | 向量化，C 级性能的数组运算 |
| ufunc | 通用函数，逐元素快速运算 |

## 快速开始

```bash
make run
```

## 依赖

- Python 3.6+
- NumPy (`pip install numpy`)

## 验证

```bash
make test
```
