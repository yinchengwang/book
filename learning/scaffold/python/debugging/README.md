# debugging - Python 调试技巧

## 概述

本卡演示 Python 调试工具：pdb、breakpoint、logging 调试。

## 核心概念

| 概念 | 说明 |
|------|------|
| print | 最简调试方式 |
| logging | 结构化日志调试 |
| pdb | Python 交互式调试器 |
| breakpoint() | Python 3.7+ 内置断点 |
| traceback | 异常堆栈追踪 |

## 快速开始

```bash
# 运行演示
make run

# pdb 交互调试
make pdb
```

## 验证

```bash
make test
```
