# python-closures 学习笔记

## 概念地图

闭包 = 函数 + 引用环境：

- **嵌套函数**：函数内部定义的函数
- **自由变量**：在内部函数中引用但未定义的变量
- **闭包**：捕获自由变量的函数，其 __closure__ 属性存储环境
- **nonlocal**：在嵌套函数中声明要修改外部变量

## 闭包 vs 类

| 特性 | 闭包 | 类实例 |
|------|------|--------|
| 语法 | 嵌套函数 | class + self |
| 数据 | 闭包变量 | 实例属性 |
| 方法 | 内部函数 | 方法函数 |
| 状态 | 不可见 | 可访问 |

## 踩坑记录

1. **循环引用**：闭包引用可变对象可能导致循环引用
2. **延迟绑定**：循环中创建闭包时使用 late binding 问题
3. **内存泄漏**：闭包可能阻止垃圾回收被捕获的变量

## 工程对照（≥100 字硬约束）

闭包在 Python 中无处不在：

1. **装饰器**：本质是返回闭包的函数
2. **工厂函数**：`make_multiplier` 模式用于创建专用函数
3. **回调函数**：事件处理中常用闭包捕获上下文
4. **偏函数**：`functools.partial` 用闭包实现
5. **私有变量**：替代类的简单场景用闭包更轻量
6. **记忆化**：缓存计算结果避免重复执行
