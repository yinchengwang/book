# python-bytecode 学习笔记

## 概念地图

Python 字节码是 CPython 执行代码的方式：

- **dis 模块**：反汇编工具，分析字节码
- **co_code**：代码对象的字节码序列
- **字节码指令**：LOAD_FAST, STORE_FAST, BINARY_ADD 等

## 常见字节码指令

| 指令 | 含义 |
|------|------|
| LOAD_FAST | 加载局部变量 |
| STORE_FAST | 存储局部变量 |
| BINARY_ADD | 加法 |
| COMPARE_OP | 比较 |
| CALL_FUNCTION | 调用函数 |
| RETURN_VALUE | 返回值 |

## 为什么学习字节码

1. **理解 Python 执行**：理解语言如何工作
2. **性能优化**：识别低效模式
3. **调试**：理解错误信息
4. **源码分析**：看别人怎么写代码

## 工程对照（≥100 字硬约束）

字节码知识在以下场景有用：

1. **性能优化**：避免创建不必要的变量
2. **理解生成器**：yield 字节码转换
3. **装饰器原理**：理解 wrapper 如何工作
4. **Python 内部**：理解 CPython 虚拟机
5. **反编译**：分析 .pyc 文件
6. **教学**：深入理解语言机制

学完本卡能动手的事：用 dis.dis() 分析自己的函数，观察字节码。
