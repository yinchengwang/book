# python-advanced-summary scaffold

Python 进阶主题总结。

## 复现命令

```bash
cd learning/scaffold/python/python-advanced-summary
python3 main.py
```

- 装饰器（decorator）
- 生成器（generator）
- 上下文管理器（context manager）
- 元类（metaclass）
- 闭包（closure）
- 异步编程（asyncio）
- 多线程（threading）
- 多进程（multiprocessing）
- 类型提示（type hints）
- 描述符（descriptor）
- __slots__
- ABC 抽象基类
- dataclasses

详见其他各模块的 README.md。
