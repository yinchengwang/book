# pandas - Pandas 数据分析

## 概述

本卡演示 Pandas 数据分析核心功能：DataFrame、筛选、聚合和清洗。

## 核心概念

| 概念 | 说明 |
|------|------|
| DataFrame | 表格数据，pandas 核心数据结构 |
| Series | 一维序列，类似字典或列表 |
| groupby | 分组聚合 |
| merge | 数据连接 |
| pivot | 数据透视表 |

## 快速开始

```bash
make run
```

## 依赖

- Python 3.6+
- pandas (`pip install pandas`)
- numpy (`pip install numpy`)

## 验证

```bash
make test
```
