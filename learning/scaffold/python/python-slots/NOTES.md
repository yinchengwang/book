# python-slots 学习笔记

## 概念地图

__slots__ 是 Python 内存优化机制：

- **__slots__**：类属性，定义允许的实例属性
- **内存节省**：避免每个实例的 __dict__ 字典
- **属性限制**：只能使用 __slots__ 中定义的属性
- **继承**：派生类也需要定义 __slots__

## __slots__ vs __dict__

| 特性 | __slots__ | __dict__ |
|------|-----------|----------|
| 内存 | 节省 ~40-60% | 较大 |
| 灵活性 | 限制属性 | 任意属性 |
| 继承 | 需显式定义 | 自动获得 |

## 踩坑记录

1. **多重继承**：多个父类都有 __slots__ 时可能有冲突
2. **pickle**：带 __slots__ 的类可以 pickle
3. **property**：配合 property 使用私有存储 `_name`

## 工程对照（≥100 字硬约束）

__slots__ 在性能敏感场景中使用：

1. **游戏引擎**：大量游戏对象，内存敏感
2. **数据类**：numpy record 用 __slots__
3. **缓存**：减少缓存对象内存
4. **高频对象**：日志对象、事件对象
5. **attrs 库**：使用 __slots__ 选项
6. **pytest fixtures**：大量测试 fixture
