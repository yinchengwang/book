# python-descriptors 学习笔记

## 概念地图

描述符是 Python 属性访问的核心机制：

- **__get__**：获取属性时调用
- **__set__**：设置属性时调用
- **__delete__**：删除属性时调用
- **数据描述符**：同时实现 get 和 set，优先级最高
- **非数据描述符**：只实现 get（函数默认）

## 描述符优先级

1. 数据描述符（实现了 `__get__` 和 `__set__`）
2. 实例字典中的值
3. 非数据描述符
4. `__getattr__`

## 踩坑记录

1. **storage_name**：避免属性名冲突，用 `_name` 存储
2. **__set_name__**：3.6+ 用于获取属性名

## 工程对照（≥100 字硬约束）

描述符在框架中广泛应用：

1. **property**：内置数据描述符
2. **Django ORM**：字段验证用描述符实现
3. **Pydantic**：数据验证用描述符
4. **延迟属性**：@Lazy 用非数据描述符
5. **日志属性**：自动记录访问日志
6. **类型检查**：描述符验证属性类型
