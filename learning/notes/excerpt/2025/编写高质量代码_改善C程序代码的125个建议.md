## 摘抄 1

1.基础数据类型问题：如数据取值范围、整数溢出与回绕、浮点数精度、数据类型转换的范围检查等。
2.数组与指针问题：指针与地址、野指针、空（null）指针、NULL指针、void指针、多级指针、指针函数与函数指针，以及数组越界与缓冲区溢出等
3.内存管理问题：内存分配、内存释放、内存越界与内存泄漏等
4.字符与字符串问题：串拷贝与内存拷贝，内存重叠与溢出，字符串查找等。
5.高效设计问题：表达式设计、算法设计与函数设计，内联函数与宏的取舍等。
6.其他杂项问题：信号处理、文件系统、断言与异常处理、内嵌汇编的使用等。

## 摘抄 2

C语言的由来：
    Algol 60 -> CPL(Combined Programming Language) -> BCPL(Basic Combined Programming Language) -> B(取自BCPL的首字母) -> New B -> C(取自BCPL的第二个字母)
