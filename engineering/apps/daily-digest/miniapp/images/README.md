# DailyDigest 微信小程序

## Tab Bar 图标

需要准备 8 个 PNG 图标文件放在 `miniapp/images/` 目录：

| 文件 | 说明 |
|------|------|
| `daily.png` | 每日 Tab（未选中） |
| `daily-active.png` | 每日 Tab（选中） |
| `discover.png` | 发现 Tab（未选中） |
| `discover-active.png` | 发现 Tab（选中） |
| `collections.png` | 收藏 Tab（未选中） |
| `collections-active.png` | 收藏 Tab（选中） |
| `mine.png` | 我的 Tab（未选中） |
| `mine-active.png` | 我的 Tab（选中） |

图标尺寸：建议 48×48px，PNG 格式，使用 #2D7FF9 作为选中色。

## 使用方式

用微信开发者工具打开 `miniapp/` 目录即可预览。
