## 2. Volcano 迭代器模型

### 2.1 核心概念

Volcano 模型是数据库执行引擎的经典架构，每个算子实现统一的迭代接口：

```c
TupleTableSlot *ExecProcNode(PlanState *pstate);
```

### 2.2 执行流程

```
客户端 → ExecQuery → ExecProcNode(SeqScan) → heap_getnext
                      ↓
                   ExecProcNode(HashJoin) → ExecProcNode(SeqScan)
                      ↓
                   ExecProcNode(Agg) → ExecProcNode(HashJoin)
                      ↓
                   返回结果
```

### 2.3 PlanState 树结构

```c
typedef struct PlanState {
    NodeTag          type;
    Plan            *plan;
    EState          *state;
    ExecProcNodeMtd  ExecProcNode;
    PlanState       *lefttree;    /* 左子树 */
    PlanState       *righttree;   /* 右子树 */
    ExprState       *qual;        /* 过滤条件 */
    TupleTableSlot  *ps_ResultTupleSlot;
} PlanState;
```

## 4. 表达式求值

### 4.1 字节码解释器

表达式树编译为字节码序列：

```
Expr: (a + b) > 0

Step 0: EEOP_SCAN_VAR(a)       → resvalue = slot[a]
Step 1: EEOP_SCAN_VAR(b)       → resvalue = slot[b]
Step 2: EEOP_ADD               → resvalue = a + b
Step 3: EEOP_CONST(0)          → resvalue = 0
Step 4: EEOP_COMPARE(>)         → resvalue = (a+b) > 0
Step 5: EEOP_DONE              → 返回
```

### 4.2 ExprEvalOp 操作码

| 操作码 | 说明 |
|--------|------|
| EEOP_SCAN_VAR | 读取扫描变量 |
| EEOP_CONST | 加载常量 |
| EEOP_ADD/SUB/MUL/DIV | 算术运算 |
| EEOP_COMPARE | 比较运算 |
| EEOP_JUMP_IF_FALSE | 条件跳转 |
| EEOP_FUNCEXPR | 函数调用 |

    execute_sql("INSERT INTO t VALUES (1, 'test')", db);
    
    QueryResult *result = execute_sql("SELECT * FROM t", db);
