# Todo App UI 优化设计

> **日期**: 2026-07-13
> **目标**: 全面优化 Todo App 前端 UI/UX 体验

## 概述

将现有 Todo App 从基础功能界面升级为精致、流畅的待办管理体验。保持 GitHub 风格深色主题，融入现代交互设计和微动效。

## 设计原则

1. **层次清晰**：卡片、面板、内容区有明确视觉层级
2. **动效克制**：动效服务于反馈，不喧宾夺主
3. **响应及时**：每个操作都有视觉反馈
4. **一致性强**：统一的间距、圆角、阴影系统

---

## 1. 视觉效果增强

### 1.1 卡片系统
- **悬浮效果**：悬停时卡片微微上浮 + 边框发光
- **渐变边框**：优先级颜色作为卡片左边框
- **关闭态**：已完成项降低对比度 + 删除线

```css
.todo-card:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 20px rgba(47, 129, 247, 0.15);
  border-color: var(--primary);
}
```

### 1.2 动画系统
- **页面过渡**：淡入淡出 200ms ease
- **列表入场**：交错动画，每项延迟 50ms
- **弹窗**：scale 0.95→1 + 淡入

### 1.3 加载状态
- **骨架屏**：数据加载时显示灰色占位块
- **按钮加载**：禁用 + spinner 图标

---

## 2. 布局与排版

### 2.1 间距系统（4px 基准）
```
xs: 4px   sm: 8px   md: 16px   lg: 24px   xl: 32px
```

### 2.2 响应式断点
```
mobile: < 768px   tablet: 768-1024px   desktop: > 1024px
```
- 移动端：侧边栏变为底部标签栏
- 平板端：侧边栏可折叠

### 2.3 看板视图
- 列宽度统一 280px
- 列间距 16px
- 空列显示占位提示

---

## 3. 交互体验

### 3.1 快捷键
| 快捷键 | 功能 |
|--------|------|
| `N` | 新建待办 |
| `/` | 聚焦搜索框 |
| `Esc` | 关闭详情面板/弹窗 |
| `↑↓` | 在列表中导航 |
| `Enter` | 查看选中项详情 |

### 3.2 空状态
- 列表为空：显示友好插图 + "还没有待办，点击右上角创建第一个"
- 搜索无结果：显示空状态插图 + 清除筛选建议

### 3.3 Toast 通知
- 位置：右下角
- 自动消失：2秒
- 支持手动关闭

---

## 4. 组件优化

### 4.1 TodoCard
- 左侧优先级色条（4px 宽）
- 悬停时显示快捷操作（完成/删除）
- checklist 进度条颜色随进度变化

### 4.2 DetailPanel
- 滑入动画（右侧滑入 300px 宽面板）
- 点击外部自动关闭
- 表头固定，内容可滚动

### 4.3 FilterBar
- 搜索框 + 筛选器折叠为一行
- 筛选器支持多选标签
- 搜索防抖 300ms

### 4.4 CreateDialog
- 模态弹窗居中
- ESC 或点击遮罩关闭
- 表单验证 + 错误提示
