# 关键特性

## 学习目标
- 了解 sample_db 的关键特性
- 掌握这些特性解决的问题和使用场景

## 核心概念

- **分区**：将大表拆分为更小的物理存储单元
- **并行查询**：利用多核 CPU 加速查询
- **扩展机制**：插件、扩展、自定义函数
